🐍 Snake Game (Python + Tkinter)

📌 Description

This is a classic Snake Game built using Python and the Tkinter library.
The player controls a snake that moves around the window, eating food to grow longer while avoiding collisions with the walls and itself.

This project was developed as part of the "100 Days of Code – Python Bootcamp by Dr. Angela Yu", with hands-on implementation and understanding of core concepts.

---

🎮 Features

* Smooth snake movement using keyboard controls
* Food spawning at random positions
* Score tracking system
* Collision detection (wall and self)
* Game over logic

---

🛠️ Tech Stack

* Python
* Tkinter (GUI)
* Turtle module

---

🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/pratiksaware/python-snake-game.git
   ```

2. Navigate to the project directory:

   ```bash
   cd python-snake-game
   ```

3. Run the game:

   ```bash
   python main.py
   ```

---

🎯 Controls

| Key | Action     |
| --- | ---------- |
| W   | Move Up    |
| S   | Move Down  |
| A   | Move Left  |
| D   | Move Right |
